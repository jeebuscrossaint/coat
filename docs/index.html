coat

A single command that themes 22+ Linux applications at once — terminals, editors, window managers, bars, launchers, and more — using Base16 color schemes.

Quick start

# 1. Build and install
make && sudo make install

# 2. Clone the scheme library (~700 schemes)
coat clone

# 3. Browse available schemes
coat list --dark

# 4. Edit your config
$EDITOR ~/.config/coat/coat.yaml

# 5. Apply to all enabled apps
coat apply

Example config

scheme: gruvbox-dark-hard
enabled:
  - fish
  - foot
  - helix
  - sway
  - waybar
  - fuzzel
  - dunst
  - gtk

font:
  monospace: "JetBrains Mono"
  sizes:
    terminal: 12
    desktop:  10
    popups:   10

opacity:
  terminal: 0.95

How it works

coat reads ~/.config/coat/coat.yaml, loads the selected Base16 scheme from ~/.config/coat/schemes/, optionally transforms it through Material You color science, then calls each enabled module's apply function. Each module writes a theme file to the application's config directory. Most modules also trigger a live reload so changes take effect immediately.
