Installation

Build coat from source and set up the scheme library.

Dependencies

LibraryPurpose
libyamlParsing coat.yaml and scheme files
libjson-cJSON generation for some modules (VSCode, chromium)
libmColor math — HCT conversions, Material You
stb_image.hWallpaper image loading — bundled, no install needed

Arch Linux

sudo pacman -S libyaml json-c

Debian / Ubuntu

sudo apt install libyaml-dev libjson-c-dev

Fedora

sudo dnf install libyaml-devel json-c-devel

Building

git clone https://github.com/jeebuscrossaint/coat.git
cd coat
make

The Makefile auto-detects your C compiler (cc, then gcc, then clang). Object files go into build/core/ and build/modules/. Compile flags: -Wall -Wextra -std=c11 -O2.

Installing

# Install to /usr/local/bin/coat
sudo make install

# Remove
sudo make uninstall

First-time setup

After installing, clone the tinted-theming scheme library. This downloads ~700 Base16/Base24 YAML schemes to ~/.config/coat/schemes/. 

coat clone

To update the scheme library later:

coat update

Then create your config:

mkdir -p ~/.config/coat
$EDITOR ~/.config/coat/coat.yaml

See Configuration for the full reference, or Commands to start browsing schemes.
