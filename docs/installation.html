Installation

Build coat from source and set up the scheme library.

Dependencies

coat is written in Rust. The only build dependency is the Rust toolchain — all library dependencies (serde, tera, serde_yaml, serde_json, walkdir, dirs, anyhow) are downloaded and compiled automatically by Cargo.

Install Rust if you don't have it:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Building

git clone https://github.com/jeebuscrossaint/coat.git
cd coat
cargo build --release

The binary is at target/release/coat (or coat.exe on Windows).

Installing

# Copy to a directory in your PATH
sudo cp target/release/coat /usr/local/bin/

# Or install via cargo directly
cargo install --path .

First-time setup

Clone the tinted-theming scheme library. This downloads ~700 Base16/Base24 YAML schemes to ~/.config/coat/schemes/. 

coat clone

To update the scheme library later:

coat update

On Linux, create a config file. On Windows you can skip this and use coat set <scheme> directly. 

mkdir -p ~/.config/coat
$EDITOR ~/.config/coat/coat.yaml

See Configuration for the full reference, or Commands to start browsing schemes.
