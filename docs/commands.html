Commands

All coat CLI commands and their options.

coat clone

Clones the tinted-theming/schemes repository to ~/.config/coat/schemes/. Run once after installing coat. 

coat clone

coat update

Pulls the latest schemes from the remote repository.

coat update

coat list

Lists all available schemes. By default prints a live color preview (RGB color swatches) next to each scheme name. 

coat list                # all schemes with color preview
coat list --dark         # dark variants only
coat list --light        # light variants only
coat list --no-preview   # skip swatches, faster output

coat search

Searches scheme names and authors. Matching is case-insensitive.

coat search gruvbox
coat search nord
coat search "one dark"
coat search morhetz   # search by author

coat set

Switches to a new scheme and applies it everywhere in one shot. This is the fastest way to change your theme without editing any files. 

coat set catppuccin-mocha
coat set nord
coat set gruvbox-dark-hard

What it does:

  • Validates the scheme exists in the library
  • Updates (or creates) ~/.config/coat/coat.yaml with the new scheme name
  • Linux: applies to all enabled modules from coat.yaml
  • Windows: themes the system accent color, dark/light mode, Windows Terminal, and VSCode via the registry — no config file required

coat apply

Reads ~/.config/coat/coat.yaml, loads the configured scheme, and calls each enabled module's apply function in sequence. 

coat apply              # all enabled apps
coat apply kitty        # single app only
coat apply sway
coat apply waybar

When targeting a single app, coat still reads the full config (for font/opacity settings) but only calls that one module.

coat docs

Prints activation instructions for a specific module to stdout.

coat docs kitty
coat docs sway
coat docs waybar

coat help

Prints a summary of available commands.

coat help

Typical workflow

# First time
coat clone

# Find a scheme you like
coat list --dark
coat search catppuccin

# Switch and apply in one command
coat set catppuccin-mocha

# Apply current scheme to a single app
coat apply kitty

# Update scheme library later
coat update
