Wallpaper & Material You

Automatically generate a color palette from your wallpaper, optionally enhanced with Material You color science.

Wallpaper extraction

Requirements

  • swww must be running (swww-daemon)
  • Wallpaper must be PNG, JPG, or WebP

Usage

# One-time: extract and apply without changing coat.yaml
coat wallpaper
coat wallpaper --no-material-you

# Permanent: extract on every coat apply
# In coat.yaml:
scheme: wallpaper
material_you: true

How it works

  1. Query wallpaper — runs swww query to get the current image path
  2. Load image — reads the file with stb_image (no external dep)
  3. Quantization — Wu's Celebi algorithm (a fast k-means variant) finds the 16 most representative colors in the image
  4. Scoring — each candidate is evaluated in HCT color space:
    • Chroma ≥ 48 required for accent slots (discards grays)
    • Tone range 30–90 preferred (avoids near-black and near-white)
    • Colors weighted by pixel frequency
  5. Palette assignment
    • base00–base07 (neutrals): darkest to lightest extracted colors
    • base08–base0F (accents): most vibrant colors, spread across hues
  6. Material You — optionally applied (see below)
  7. Apply — all enabled modules are themed with the palette
Tip: Run coat wallpaper whenever you change your wallpaper. Or wire it into your wallpaper changer script so theming updates automatically.

Material You

Material You is Google's color system from Android 12+, built on the HCT (Hue-Chroma-Tone) color space. coat implements the core transformation to make any palette — from a named scheme or from wallpaper extraction — more vibrant and cohesive.

HCT color space

AxisRangeMeaning
H (Hue)0–360°Position on the color wheel
C (Chroma)0–150Colorfulness / saturation
T (Tone)0–100Perceptual lightness (0 = black, 100 = white)

HCT is perceptually uniform — equal numeric changes produce equal perceived differences regardless of hue. This is why Material You produces more natural-looking palettes than simple HSL or RGB manipulation.

What the transformation does

Accent colors (base08–base0F):

  • Dark theme: tone → 80 (bright, vibrant)
  • Light theme: tone → 40 (saturated but legible on light background)
  • Chroma → 80–120 (highly saturated)
  • Optional hue harmonization: 15% blend toward the seed hue, tying all accents together

Neutral colors (base00–base07):

  • Chroma → 2–6 (near-gray, with a subtle tint toward the seed hue)
  • Tone stepped evenly from darkest to lightest to preserve hierarchy

Enable it

material_you: true

Works with both wallpaper extraction and named schemes. When used with a named scheme, hues are preserved but tones and chroma are adjusted to the Material You targets.

Gray or near-monochrome wallpapers will produce a near-neutral palette even with Material You enabled — there's no seed hue to work with. Try a wallpaper with at least one saturated color for best results.
